

##  📝 개요
이 저장소는 구름톤 딥다이브 백엔드 1기 과정 중 수행한 실습, 과제, 그리고 개인 학습 내용을 기록하고 있습니다.  
프론트엔드와 백엔드의 여러 기술 스택을 다뤄보며, 실제 프로젝트에 적용하고 이론적인 이해를 깊이 함으로써 실무 역량을 키우는 것을 목표로 하고있습니다.
